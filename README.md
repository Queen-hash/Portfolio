## 🚀 Overview

This portfolio is designed with a "Mobile-First" approach and features a dark, moody aesthetic with neon accents (`#cdea12`). Instead of relying on heavy frameworks or libraries, this project was built using purely **Vanilla HTML, CSS, and JavaScript** to demonstrate a strong understanding of core web fundamentals, DOM manipulation, and performance optimization.

## ✨ Key Features & Technical Highlights

* **Interactive Splash Screen:** A custom-built, smooth-typing entry animation that seamlessly transitions into the main content, providing a premium initial user experience.
* **Cinematic Visuals:** Implemented subtle CSS-based film grain (SVG noise filter) and vignette effects to give the interface a deep, textured, and high-end feel.
* **Advanced Scroll Interactions:** * A dynamic top scroll progress bar tracking user reading depth.
  * Parallax scrolling effects on the profile image to create a 3D depth illusion.
  * Intersection Observer API utilized for performant, scroll-triggered reveal animations.
* **Custom UI Elements:** Completely restyled browser scrollbars to match the dark/neon theme, moving away from default OS styles.
* **Fully Responsive:** Custom mobile navigation (Hamburger Menu) using CSS transitions and JavaScript toggles, ensuring perfect usability across all device widths.
* **Dynamic Content:** Automated footer year updating via JavaScript Date objects to ensure zero-maintenance timestamping.

## 🛠️ Tech Stack

* **HTML5:** Semantic structuring and accessibility best practices.
* **CSS3:** Custom properties (variables), Flexbox, CSS Grid, media queries, and advanced keyframe animations.
* **JavaScript (ES6+):** Vanilla JS for DOM manipulation, event handling, and logic routing without external dependencies.

## 📂 Project Structure

- `index.html` — The main entry point and structural skeleton.
- `style.css` — Global styles, utility classes, and custom animations.
- `script.js` — Core interactivity, scroll monitoring, and UI toggles.
